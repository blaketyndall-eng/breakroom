## Object rules

Objects should not be random props. They should feel like evidence.

Each object should carry at least two contradictory signals:

```txt
sacred + cheap
luxury + damaged
corporate + haunted
funny + played straight
sports relic + spiritual relic
motel object + corporate record
cash object + religious object
old web file + physical artifact
```

## Product truth

The Breakroom gets stronger when objects remember more than people.

Do not explain the mythology too much. File it. Stamp it. Let the user find it twice.
